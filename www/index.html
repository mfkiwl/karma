Welcome to the Karma home page. The library has recently been converted to proper HTML, so it should look much nicer. I welcome any input on this new look. Thanks to Ariel Internet Services for the automated HTML generation.

Karma is a toolkit for interprocess communications, authentication, encryption, graphics display, user interface and manipulating the Karma network data structure. It contains KarmaLib (the structured libraries and API) and a large number of modules (applications) to perform many standard tasks.

This library is used in the ATNF Visualisation software (named Sutra) which has been developed by Tom Oosterloo (toosterl@atnf.csiro.au) and myself.

For a much more detailed description of the Karma Library, read the Karma Library Readme file.

To get your very own copy of the Karma library, check out the Karma ftp site

If you have the Karma HTML documentation files installed under /usr/local/karma/www then click here to use that copy. It will be much faster.
